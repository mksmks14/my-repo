The CSS hierarchy for every web page looks like this:

  1. The browser’s default stylesheet
  2. User-defined stylesheets
  3. External stylesheets (that’s us)
  4. Page-specific styles (that’s also us)
  5. Inline styles (that could be us, but it never should be)
